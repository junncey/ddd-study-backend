# 登录页面 UI 优化

## 修改日期
2026-02-18

## 修改类型
- style - UI 样式优化

## 修改内容

### 1. 去除测试账号文字
- 删除了原来的"测试账号：admin / admin123"提示
- 替换为动态引导文字："首次使用？立即注册"（登录页）/ "已有账号？去登录"（注册页）

### 2. 优化页面交互样式

#### 输入框交互
- 添加悬停效果：边框颜色变化、阴影效果
- 添加聚焦效果：边框高亮、阴影加深
- 输入框背景色优化
- 图标颜色随状态动态变化

#### 按钮交互
- 添加渐变背景效果
- 添加悬停提升动画（transform translateY）
- 添加动态阴影效果
- 添加点击按下动画

#### Tabs 样式
- 优化下划线样式（圆角、高度）
- 添加悬停颜色变化
- 去除默认边框线

#### 品牌区域
- 添加旋转光晕动画效果
- Logo 添加浮动动画
- 特性列表添加悬停平移效果
- 优化渐变背景色

#### 卡片头部
- 标题和副标题根据当前 Tab 动态变化
- 添加淡入下拉动画

### 3. 新增动画效果
- `fadeInDown` - 淡入下拉动画
- `shimmer` - 光晕旋转动画
- `float` - 浮动动画

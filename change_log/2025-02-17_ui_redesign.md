# 前端UI重新设计

## 日期
2025-02-17

## 概述
对整个电商平台前端UI进行了重新设计，采用现代简约风格，实现了Web端和手机端的响应式适配。

## 设计方向

### 美学风格
- **风格**: 现代简约、优雅精致
- **特点**: 清爽的白色背景、温暖的橙色强调色、大量留白、精致细节

### 设计系统

#### 配色方案
- **主色**: `#FF6B35` - 温暖橙色（激发购买欲）
- **主色浅**: `#FF8C61` - hover状态
- **主色深**: `#E55A2B` - 按压状态
- **背景主色**: `#FFFFFF` - 纯白
- **背景次色**: `#FAFAFA` - 淡灰白
- **文字主色**: `#1A1A2E` - 深蓝黑
- **文字次色**: `#6B7280` - 中灰

#### 字体
- **标题**: Outfit - 现代几何感的无衬线字体
- **正文**: Noto Sans SC - 优雅的中文无衬线字体

#### 圆角与阴影
- **圆角**: 8px/12px/16px/24px 四级圆角系统
- **阴影**: 轻微阴影，层次分明

## 响应式断点

| 断点 | 宽度 | 设备 |
|------|------|------|
| xs | < 576px | 手机 |
| sm | 576px - 767px | 大屏手机 |
| md | 768px - 991px | 平板 |
| lg | 992px - 1199px | 小屏桌面 |
| xl | ≥ 1200px | 大屏桌面 |

## 移动端特性

1. **底部导航栏**: 固定在屏幕底部，包含主要功能入口
2. **抽屉菜单**: 右侧滑出，包含完整菜单
3. **固定操作栏**: 商品详情、购物车等页面的操作按钮固定在底部
4. **卡片式布局**: 列表页面采用卡片式布局替代表格
5. **安全区域适配**: 适配iPhone X系列底部安全区域

## 动画效果

1. **fadeIn**: 页面淡入
2. **fadeInUp**: 向上淡入
3. **scaleIn**: 缩放淡入
4. **商品卡片悬停**: 上移4px + 阴影增强
5. **图片悬停**: 1.05倍缩放
